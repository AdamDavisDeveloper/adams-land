++++++++++++++++++++++++++++++++++++++++++++++++++++++++
>>> VISUAL NOVEL ENGINE
++++++++++++++++++++++++++++++++++++++++++++++++++++++++
info: This will be an lightweight engine for creating visual-novel
style content on the web. Things to include:
1. A text-box section down below for character speech
2. A background image with config properies like Blur/darken/etc.
   which can be set for each "scene".
3. A timeline creator for each "scene" which is a collection of "pages"
4. A fuction (or something) for calling a scene which renders the component
   with the appropriate scene or array of scenes.
5. Each scene will be configured with: character image(s) + orientation (left/right), character text (one per page),
   background image (default black), typing/talking speed, player actions (default is "next-page"),
   chat/text orientation (left/right side - switch side for each currently speaking character),
   chat/text background (just a few options in case the background doesn't match the text box), and 
   music

This will be a great way to have people engage with characters in Adamsland. :)
